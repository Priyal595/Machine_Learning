## What is Machine Learning?

Machine Learning (ML) is a field of computer science where computers learn patterns from data and use those patterns to make predictions, decisions, or generate content without being explicitly programmed.

### Definition by Tom Mitchell

Tom Mitchell, a professor at Carnegie Mellon University, defined Machine Learning as:

> **"A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P if its performance at tasks in T, as measured by P, improves with experience E."**

This definition explains three important components of Machine Learning:

- **Experience (E):** The data the model learns from  
- **Task (T):** The specific job the model performs (e.g., classification, prediction)  
- **Performance Measure (P):** The metric used to evaluate how well the model performs

### Simple Definition

In basic terms, **Machine Learning is the process of training a piece of software, called a model, to learn patterns from data and use those patterns to make predictions or generate content such as text, images, audio, or video.**

### Examples of Machine Learning Applications

- Email spam detection  
- Movie recommendation systems  
- Face recognition in smartphones  
- Voice assistants like Alexa or Siri  
- Generative AI tools that create text, images, or videos
